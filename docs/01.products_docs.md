# 상품 목록 페이지 프로젝트

## 프로젝트 개요

상품 목록 페이지 - 고객에게 다양한 상품을 표시하고, 상품을 선택하여 장바구니에 추가할 수 있는 기능을 제공합니다.

## 개발 환경

React 17.0, JavaScript ES6

## 설치 및 실행 방법

```
npm install
npm start
```

## 컴포넌트 구조

예시:

- `ProductList`: 상품 목록을 보여줍니다.
- `ProductCard`: 각 상품의 정보를 보여줍니다.

## 테스트 방법

```

```
